# Interview Trainer

Приложение для подготовки к техническим собеседованиям. Оффлайн, структурированно, с прогрессом.

## О проекте

Interview Trainer помогает системно готовиться к собеседованиям по Android разработке (в будущем — и другие направления). Можно учить теорию, проверять себя билетами и отслеживать прогресс.

## Архитектура

Проект построен на принципах Clean Architecture:

- presentation — UI экраны, ViewModel, навигация (Jetpack Compose)
- domain — модели и интерфейсы репозиториев
- data — реализация репозиториев, Room база данных

## Стек технологий

- Kotlin — основной язык
- Jetpack Compose — UI
- Room — локальная база данных
- Flow / StateFlow — реактивные потоки
- Coroutines — асинхронность
- Navigation Compose — навигация
- Clean Architecture — структура проекта

## Возможности (в разработке)

- Выбор платформы (Android, iOS, Python ML, Frontend)
- Раздел теории с сортировкой от простого к сложному
- Билеты с вопросами
- Тесты и развернутые ответы
- Статистика прогресса
- Избранное
